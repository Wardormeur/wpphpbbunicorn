== Description ==

This plugin allows for the integration of Symfony2 components into Wordpress. PHP5.3 is desired but not mandatory.

I advise you strongly read this [article] (http://www.craftitonline.com/2011/09/wordpress-symfony2-powered-the-ultimate-wp-plugin/) before posing any question.

== Installation ==

You can use git to fetch the symfony component libraries or you can also just download them by hand and copy them under the lib folders.

1. Upload `wp-sf2.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Run the commands
`
cd wp-content/plugins/
git clone git://github.com/cordoval/wp-sf2.git
git submodule init
git submodule update
`
